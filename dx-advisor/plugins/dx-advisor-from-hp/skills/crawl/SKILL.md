# 企業Webサイト情報収集

対象URL：**$ARGUMENTS**

---

## 実行手順

### Step 1：web-researcherエージェントチームの並列起動

以下の3つのweb-researcherエージェントを **並列で** 起動してください。
各エージェントには対象URLと調査観点を明確に指示してください。

#### エージェント1：企業概要・事業構造の収集
```
対象URL: $ARGUMENTS
ミッション: このURLのホームページを起点に、企業概要（会社名、業種、従業員数、売上規模、設立年、拠点）と事業構造（主力事業、ビジネスモデル、顧客セグメント、バリューチェーン）を収集してください。
会社概要ページ、事業紹介ページ、IR情報ページを重点的に調査してください。
```

#### エージェント2：業務プロセス・組織体制の収集
```
対象URL: $ARGUMENTS
ミッション: このURLのホームページを起点に、業務プロセスと組織体制を調査してください。
採用ページ（募集職種から業務内容を推定）、サービス提供フロー、お問い合わせ対応フロー、
ニュース・プレスリリース（最新の取り組み）を重点的に調査してください。
主要な部門・職種・業務フローを整理してください。
```

#### エージェント3：IT/DX状況・業界動向の調査
```
対象URL: $ARGUMENTS
ミッション: このURLの企業について、既存のIT活用状況とDX/AI関連の取り組みを調査してください。
サイトの技術的な特徴（ECサイト、予約システム、マイページ等のデジタルサービスの有無）を確認し、
Web検索で「{企業名} DX」「{企業名} AI 導入」「{企業名} デジタル化」「{企業名} 業界 トレンド」を調査してください。
```

### Step 2：収集結果の統合

3つのエージェントの結果を統合し、以下の形式でまとめてください。
