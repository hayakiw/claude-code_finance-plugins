# DX/AI導入効果分析 — フルアセスメント

対象URL：**$ARGUMENTS**

---

このスキルは3フェーズ（情報収集→分析→レポート作成）を専門エージェントチームで実行し、
高品質なDX/AI導入提案レポートを自動生成します。

---

## Phase 1：情報収集（web-researcherエージェントチーム）

3つのweb-researcherエージェントを **並列起動** し、対象企業の情報を網羅的に収集します。

### エージェント1：企業概要・事業構造
web-researcherエージェントを起動：
```
対象URL: $ARGUMENTS
ミッション: このURLのホームページを起点に、企業の全体像を調査してください。
以下を重点的に収集：
- 企業プロフィール（企業名、業種、従業員数、売上規模、設立年、拠点）
- 事業構造（主力事業、売上構成、ビジネスモデル、顧客セグメント）
- バリューチェーンの特徴
会社概要、事業紹介、IR情報のページを優先的に巡回してください。
サイトのナビゲーションから到達できる主要ページのリンクも収集してください。
```

### エージェント2：業務プロセス・組織体制
web-researcherエージェントを起動：
```
対象URL: $ARGUMENTS
ミッション: このURLの企業の業務プロセスと組織体制を調査してください。
以下を重点的に収集：
- 主要な業務プロセス（営業、製造/サービス提供、物流、サポート等）
- 組織体制・部門構成（採用ページの募集職種から推定）
- サービス提供フロー、顧客対応フロー
- 最新のニュース・プレスリリース（直近1年の重要なもの）
採用ページ、サービスフロー、ニュースページを優先的に巡回してください。
```

### エージェント3：IT/DX状況・業界調査
web-researcherエージェントを起動：
```
対象URL: $ARGUMENTS
ミッション: このURLの企業のIT活用状況とDX/AIの取り組み、業界動向を調査してください。
以下を重点的に収集：
1. サイトから読み取れるデジタルサービス（EC、予約システム、マイページ、チャットボット等）
2. Web検索で企業のDX/AI関連情報を調査：
   - "{企業名}" DX デジタルトランスフォーメーション
   - "{企業名}" AI 導入 活用
   - "{企業名}" 業務効率化 自動化
   - "{企業名}" IT投資 システム
3. 業界のDX/AIトレンドと課題：
   - "{業界名}" DX 成功事例 AI
   - "{業界名}" 課題 トレンド {現在の年}
```

**3つのエージェントの結果を統合し、企業プロフィール表を作成してください。**

---

## Phase 2：DX/AI機会分析（dx-analystエージェントチーム）

Phase 1の結果を元に、2つのdx-analystエージェントを **並列起動** します。

### エージェント4：フロントオフィス・顧客接点のDX/AI分析
dx-analystエージェントを起動：
```
以下の企業情報を分析し、フロントオフィス（営業・マーケティング・カスタマーサポート・顧客接点）領域でのDX/AI導入機会を特定してください。

【企業情報】
（Phase 1の統合結果を貼り付け）

以下の観点で各機会を評価（各5点、計20点）：
- 業務インパクト（売上向上・コスト削減への貢献度）
- AI適合性（現在のAI技術で対応可能な度合い）
- 実現容易性（データ整備状況・導入ハードル）
- 緊急性（業界トレンド・競合状況からの優先度）

適用可能なAI技術カテゴリ：
1. 生成AI（LLM）— 文書作成、チャットボット、ナレッジ検索
2. 予測AI — 需要予測、顧客離反予測、価格最適化
3. 画像認識 — OCR、外観検査
4. RPA — データ入力自動化、ワークフロー自動化
5. データ分析/BI — ダッシュボード、顧客分析

同業界のDX/AI成功事例もWeb検索で調査し、提案の根拠としてください。
```

### エージェント5：バックオフィス・コア業務のDX/AI分析
dx-analystエージェントを起動：
```
以下の企業情報を分析し、バックオフィス（経理・人事・総務・法務）とコア業務（製造/サービス提供・物流・研究開発）領域でのDX/AI導入機会を特定してください。

【企業情報】
（Phase 1の統合結果を貼り付け）

以下の観点で各機会を評価（各5点、計20点）：
- 業務インパクト（売上向上・コスト削減への貢献度）
- AI適合性（現在のAI技術で対応可能な度合い）
- 実現容易性（データ整備状況・導入ハードル）
- 緊急性（業界トレンド・競合状況からの優先度）

適用可能なAI技術カテゴリ：
1. 生成AI（LLM）— 文書作成、契約書レビュー、社内FAQ
2. 予測AI — 在庫最適化、異常検知、品質管理
3. 画像認識 — 検査自動化、帳票OCR
4. RPA — 定型業務自動化、システム間連携
5. データ分析/BI — 経営ダッシュボード、KPIモニタリング

同業界のDX/AI成功事例もWeb検索で調査し、提案の根拠としてください。
```

**2つのエージェントの結果を統合し、全施策をスコア降順でランキングしてください。**

---

## Phase 3：レポート作成（report-writerエージェント）

Phase 1（情報収集）とPhase 2（分析）の全結果を統合し、report-writerエージェントを起動します。

### エージェント6：最終レポート作成
report-writerエージェントを起動：
```
以下の情報収集結果と分析結果を統合し、経営層向けのDX/AI導入提案レポートを作成してください。

【企業情報（Phase 1）】
（Phase 1の統合結果）

【DX/AI機会分析（Phase 2）】
（Phase 2の統合ランキング）

以下の構成で完全なレポートを作成してください：

1. エグゼクティブサマリー（1分で把握できる3〜5文の要約）
2. 企業概要と現状分析（デジタル化レベル★5段階評価含む）
3. DX/AI導入機会マップ（バリューチェーン上のマッピング）
4. 重点提案TOP5（各施策の詳細：課題→提案→効果→ロードマップ→リスク）
5. DX推進ロードマップ（短期0-6ヶ月/中期6-18ヶ月/長期18-36ヶ月）
6. 投資対効果（ROI概算表）
7. 成功に向けた提言（推進体制、リスク、次のステップ）

重要：
- 各提案には必ず定量的な効果予測を含めること
- 企業の規模・業種に合った現実的な提案にすること
- 業界の成功事例を根拠として引用すること
```

---

## 最終出力

report-writerエージェントの出力をそのまま最終レポートとして表示してください。
レポートの冒頭に以下を付記：

---
**DX/AI導入効果分析レポート**
対象：$ARGUMENTS
分析日：（本日の日付）
エージェントチーム：web-researcher ×3 + dx-analyst ×2 + report-writer ×1

---

> **免責事項**：本レポートはAIによる自動分析です。公開情報から推定した内容を含むため、
> 実際の業務内容や課題とは異なる場合があります。具体的な導入検討にあたっては、
> 詳細なヒアリングとPoC（概念実証）を推奨します。
