# DX/AI導入レポート作成

対象企業情報：**$ARGUMENTS**

---

## 実行手順

### Step 1：report-writerエージェントの起動

report-writerエージェントを起動し、これまでの会話で収集・分析された情報を全て渡してください。

エージェントへの指示：
```
以下の情報収集結果と分析結果を統合し、経営層向けのDX/AI導入提案レポートを作成してください。
レポートテンプレートに従い、全セクションを漏れなく記載してください。

【情報収集結果】
（crawlスキルの出力を貼り付け）

【分析結果】
（analyzeスキル相当の分析結果を貼り付け）
```

### Step 2：レポート品質チェック

report-writerの出力に対して以下を確認・補完：

1. **具体性チェック**：各提案に定量的な効果予測があるか
2. **実現性チェック**：ロードマップが現実的か
3. **整合性チェック**：企業の規模・業種に合った提案か
4. **網羅性チェック**：重要な業務領域を見落としていないか

### Step 3：最終レポート出力

品質チェックを反映した最終レポートを出力してください。

---

## レポート構成（必須セクション）

1. **エグゼクティブサマリー** - 経営層が1分で把握できる要約
2. **企業概要と現状分析** - 現在のデジタル化レベル評価
3. **DX/AI導入機会マップ** - バリューチェーン上の機会の全体像
4. **重点提案（TOP 5）** - 具体的な施策、効果、ロードマップ
5. **DX推進ロードマップ** - 短期/中期/長期の実行計画
6. **投資対効果（ROI概算）** - 投資額と期待効果の概算
7. **成功に向けた提言** - 推進体制、リスク、次のステップ

---

> このスキルは `/dx-advisor:assess` の最終フェーズで自動的に呼ばれます。
> 分析結果が既にある場合、単独でレポート作成のみ実行することもできます。
