# 銘柄スクリーニング

スクリーニング対象：**$ARGUMENTS**

（例）
- `日経225` — 日経平均構成銘柄全体
- `IT・テクノロジーセクター` — セクター指定
- `7203, 9984, 6758, 4661, 8306` — カンマ区切りの銘柄コード

## ステップ1：データ取得

`financial-data` MCPサーバーが利用可能な場合は `screen_stocks` ツールで一括データ取得してください。
利用できない場合は、各銘柄についてWeb検索で財務データを調査してください。

## ステップ2：2段階スクリーニングフィルター

### Pass 1 ── 品質フィルター（企業の質）
以下をすべて満たす銘柄のみ通過：

| 指標 | 基準値 | 意図 |
|------|--------|------|
| ROE | ≥ 15% | 株主資本を効率的に活用 |
| 営業利益率 | ≥ 10% | 本業の収益力 |
| 自己資本比率 | ≥ 40% | 財務健全性 |
| 営業CF | 直近3期すべて黒字 | 実際の現金創出力 |
| EPS成長率（3年平均） | ≥ 5% | 株主価値の持続的向上 |

### Pass 2 ── バリュエーションフィルター（割安度）
以下のいずれかを満たす：

| 指標 | 基準値 |
|------|--------|
| PER | ≤ 25倍（または過去5年平均より10%以上割安） |
| PBR | ≤ 3倍 |
| 配当利回り | ≥ 1.5%（配当銘柄のみ適用） |
