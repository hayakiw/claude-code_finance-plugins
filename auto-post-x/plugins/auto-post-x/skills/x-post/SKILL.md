# X自動投稿スキル

x-composeスキルで作成済みの投稿文を受け取り、Chromeブラウザ経由でXに自動投稿する。
ユーザーへの確認は不要で、投稿まで自動完了する。

## 投稿手順

1. `tabs_context_mcp` でタブ情報を取得（タブがなければ `createIfEmpty: true` で作成）
2. タブで `https://x.com/compose/post` に移動
3. 3秒待ってスクリーンショットを撮り、ログイン状態を確認
4. **ログイン済みの場合：**
   - 投稿フォーム（「いまどうしてる？」のエリア）をクリック
   - 投稿テキストを `type` アクションで入力
   - ハッシュタグのサジェストが出たら `Escape` キーで閉じる
   - 「ポストする」ボタンをクリック
   - 2秒待ってスクリーンショットを撮り、投稿成功を確認
5. **ログインしていない場合：**
   - ユーザーに「Xにログインしてから再実行してください」と通知して中断

## エラー時の対応

- 投稿ダイアログが表示されない場合：ページをリロードして再試行
- アカウント確認ダイアログが出た場合：「OK」をクリックして進める
- それ以外のエラー：スクリーンショットを撮ってユーザーに報告
