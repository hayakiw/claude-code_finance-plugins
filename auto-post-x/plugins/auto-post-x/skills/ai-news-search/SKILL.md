# AI最新情報 検索・収集スキル

AI業界の最新ニュースをWeb検索で収集し、仕事で役立つ1トピックを選定して要点をまとめる。

## 対象トピック

以下のジャンルから、直近で最も注目度が高く仕事に活かせるニュースを1つ選ぶ：

- Claude Code の新機能・アップデート（Code Review、Agent Teams、Compaction、Coworkなど）
- OpenAI Codex CLI の新機能・アップデート
- Gemini の新機能・アップデート
- AIエージェント構築に関する最新ツール・フレームワーク
- AIの業務活用・仕事効率化の実践的な事例やツール

## 検索手順

### ステップ1：複数クエリで検索

WebSearchツールで以下のようなクエリを複数試し、鮮度の高い情報を集める。

検索キーワード例：
- 「Claude Code new feature 2026」
- 「Claude Code update」
- 「OpenAI Codex CLI update」
- 「Gemini update 2026」
- 「AI agent framework new」
- 「AI 業務効率化 最新ツール」

### ステップ2：1トピック選定

検索結果から1つだけトピックを選ぶ。選定基準：
- 鮮度が高い（直近数日以内が理想）
- 仕事で使える実践的な情報である
- エンジニア・ビジネスパーソンにとって有益で興味を引く内容である
- 具体的な数値データや機能の詳細が含まれている（投稿で🔹ポイントにしやすい）

### ステップ3：要点の整理

選定したトピックについて、以下の項目を整理する：

- **タイトル**：機能名や製品名を含む一行の見出し
- **概要**：何がどうすごいのか、インパクトのある1〜2文
- **ポイント3つ**：具体的な数値や機能の詳細（🔹で箇条書きにできる粒度）
- **締め**：仕事への影響や今後の展望
- **情報源URL**（あれば）
- **関連ハッシュタグ候補**

整理が完了したら、x-composeスキルに進む。
