# X投稿文作成スキル

ai-news-searchスキルでまとめた要点をもとに、X投稿文を作成する。
X無料アカウントの文字数制限（280 weighted characters = 日本語約140文字）に必ず収めること。
作成のみを担当し、実際の投稿はx-postスキルに渡す。

## 文字数制限について

Xの文字カウントルール：
- 日本語・CJK文字 = 2カウント
- 英数字・記号 = 1カウント
- 絵文字 = 2カウント
- 合計280カウント以内（日本語のみなら約140文字）

投稿文は必ず **120文字以内（日本語換算）** を目安に作成し、余裕を持たせること。

## 投稿フォーマット

改行なしの一段落形式で作成する：

```
[機能名/製品名]が[何をした]！[具体的な内容や数値]。[仕事への影響] #ハッシュタグ1 #ハッシュタグ2 #エンジニアと繋がりたい
```

## 作成ルール

- **1トピック1投稿が鉄則。** 複数の話題を混ぜない
- 冒頭に具体的な機能名・製品名を入れる
- 何がすごいのか、仕事でどう使えるかを1〜2文で凝縮
- 具体的な数値やコマンド例があれば含める
- ハッシュタグは2〜3個。必ず「#エンジニアと繋がりたい」を含める
- 絵文字は使わないか最小限（文字数を食うため）
- トーン：情報提供型、専門的すぎず読みやすく
- **改行は使わない**（文字数を節約するため一段落で書く）

## 良い投稿の例

```
Claude Codeに/loopコマンド追加！プロンプトの定期実行が可能に。/loop 5mでデプロイ監視を5分間隔で自動化、cron対応で柔軟なスケジュールも。開発の監視自動化が劇的に楽になる #ClaudeCode #AI開発 #エンジニアと繋がりたい
```

```
Gemini 3.1 Flash-Liteがプレビュー公開。軽量高速でモバイルアプリへのAI組み込みが簡単に。API呼び出しコスト約60%削減、オンデバイス推論にも対応予定 #Gemini #AI開発 #エンジニアと繋がりたい
```

```
OpenAI Codex CLIがv0.114に更新。GPT-5.4ネイティブ対応でコード生成精度が向上、マルチファイル一括リファクタリングも可能に #Codex #AI開発 #エンジニアと繋がりたい
```

## 悪い投稿の例

```
❌ Claude Codeが進化！一方Codexも更新、さらにGeminiも… ← 複数話題を混ぜている
```

```
❌ 【速報】Claude Codeに新機能登場！🚀\n\n概要…\n🔹ポイント1\n🔹ポイント2\n🔹ポイント3\n締め ← 改行・絵文字多用で文字数オーバー
```

投稿文が完成したら、x-postスキルに渡してXに投稿する。
